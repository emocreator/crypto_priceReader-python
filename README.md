# Crypto Scraper

## Program needs to run with env variables

### On windows run:

`set api_key={YOUR_API_KEY}`

### On Linux run:
`export api_key={YOUR_API_KEY}`


### Requirements:
### (You can create a venv, up to your decision)

`pip install -r requirements.txt`

### Execute the main file right after it:
`python main.py`
